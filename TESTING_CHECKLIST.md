# E. 비즈니스 흐름 E2E (curl)

## E-1. ID 확인
```bash
curl -s http://localhost:9080/api/auth/companies/type/BP_COMPANY | python3 -c "import sys,json; [print(f'{c[\"name\"]}: {c[\"id\"]}') for c in json.load(sys.stdin)]"
curl -s http://localhost:9080/api/auth/companies/type/EQUIPMENT_SUPPLIER | python3 -c "import sys,json; [print(f'{c[\"name\"]}: {c[\"id\"]}') for c in json.load(sys.stdin)]"
curl -s http://localhost:9080/api/dispatch/sites | python3 -c "import sys,json; [print(f'{s[\"name\"]}: {s[\"id\"]}') for s in json.load(sys.stdin)]"
```

## E-2. 현장 생성 (원형)
```bash
curl -X POST http://localhost:9080/api/dispatch/sites -H "Content-Type: application/json" \
  -d '{"name":"E2E현장","address":"서울","bpCompanyId":"<BP_ID>","boundaryType":"CIRCLE","centerLat":37.5,"centerLng":127.0,"radiusMeters":300}'
```
- [ ] 201 Created + centerLat/centerLng/radiusMeters 확인

## E-3. 현장 생성 (폴리곤)
