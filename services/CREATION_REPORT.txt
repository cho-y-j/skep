DISPATCH SERVICE Features:
  1. Deployment Plan Management
     - Create long-term deployment plans
     - Flexible rate structures (daily, overtime, night, early-morning, etc.)
     - Status tracking (ACTIVE, EXTENDED, REDUCED, COMPLETED)

  2. Daily Roster Management
     - Driver and guide assignment
     - Submit and approval workflow
     - Status tracking (PENDING, APPROVED, REJECTED)

  3. Work Record Management
     - GPS-based clock-in verification
     - Work start/end tracking
     - Multiple work type support
     - Location-based recording

  4. Work Confirmation Management
     - Automatic daily confirmation generation
     - Monthly summary confirmations
     - Electronic signature support
     - Status workflow (DRAFT → SIGNED → SENT)

INSPECTION SERVICE Features:
  1. Safety Inspection Management
     - GPS distance validation (50-meter maximum)
     - Enforced item sequence (sequence_number validation)
     - Timestamp-based recording (server time)
     - Photo attachment support
     - Status tracking (IN_PROGRESS → COMPLETED/FAILED)

  2. Maintenance Inspection Management
     - Daily driver/equipment inspections
     - Multi-component fluid status tracking
     - Mileage recording
     - Historical data maintenance

  3. Inspection Item Management
     - Equipment type-specific items
     - Active/inactive item management
     - Flexible sort ordering
     - Inspection method documentation

4. GPS Validation
     - Haversine formula implementation
     - Earth radius: 6,371,000 meters
     - Allowed range: 50 meters
     - Real-time distance calculation
     - Error messaging for violations

================================================================================
                          DATABASE SCHEMA
================================================================================

DISPATCH SERVICE (5 Tables):
  1. deployment_plans
     - UUID ID, supplier/BP company references
     - Date/time ranges, multiple rate types
     - Status and notes columns
     - Indexes on supplier, equipment, status

  2. daily_rosters
     - UUID ID, plan reference, work date
     - Driver and guide IDs (JSONB array)
     - Submission and approval tracking
     - Status management

  3. work_records
     - UUID ID, roster reference
     - Worker type and ID
     - Clock-in with GPS location (POINT type)
     - Work timing and type tracking

  4. monthly_work_confirmations
     - UUID ID, plan reference
     - Year-month identifier
     - Aggregated work hours and counts
     - Total amount calculation
     - Signature and send tracking

  5. daily_work_confirmations
     - UUID ID, work record reference
     - Site and company information
     - Work details (content, location, times)
     - Signature tracking

INSPECTION SERVICE (4 Tables):
  1. inspection_item_masters
     - UUID ID, equipment type reference
     - Item number with uniqueness constraint
     - Inspection method description
     - Photo requirement flag
     - Sort ordering

  2. safety_inspections
     - UUID ID, equipment and inspector references
     - Inspection date and timing
     - GPS coordinates (inspector and equipment)
     - Calculated distance
     - Status and notes

  3. inspection_item_results
     - UUID ID, inspection and item references
     - Result value (OK, NG, NA)
     - Photo URL and notes
     - Sequence number (order enforcement)
     - Record timestamp

  4. maintenance_inspections
     - UUID ID, equipment and driver references
     - Inspection date
     - Fluid levels (oil, coolant, hydraulic)
     - Fuel level and mileage
     - Notes and timestamp

   $ docker-compose logs -f
