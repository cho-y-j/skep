문제 해결 가이드
================================================================================

1. 포트 충돌:
   - lsof -i :8080  (포트 확인)
   - kill -9 <PID>  (프로세스 종료)

2. 데이터베이스 연결:
   - psql -h localhost -U postgres -d skep_auth

3. Redis 연결:
   - redis-cli ping

4. 빌드 실패:
   - ./gradlew clean build --stacktrace
